Order Views¶

class order.views.AllOrdersView(**kwargs)[source]¶

View to show all the past orders. This view also features a filter i.e. if you want to see all the coffee orders which are of tall and venti size.

get(request, *args, **kwargs)[source]¶

Will show all orders placed till date and total sales and also renders a form to filter by type and/or size.

post(request, *args, **kwargs)[source]¶

Filteration view. If user wants to filter all tall coffee orders or all tall and venti tea orders this View makes that happen.

class order.views.GroupedOrder(**kwargs)[source]¶

Parent view for grouping orders by type or size, in future if we want to add different way of grouping this will help user achieving the same without duplicating any code, see the views GroupByType and GroupBySize views.
